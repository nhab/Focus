Welcome to Focus.js

Focus.js is a framework for web development which have changed

many common practices to be more efficient.

Philosophy

there are a bunch of phylosophies and practices behind the focus.js including

  • Focusing on one skill (javaScript) instead of lots of skills

  • separation of real concerns

    the real concerns in front-end part are:

    struct : structuring the page

    state : Initiating and maintaining the state of page`s elements during interactions.

    presentation: how we present all these to users,including positioning and look and feel.
